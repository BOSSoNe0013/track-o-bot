# Build Dependencies

* Qt 4.8
* (OSX) Sparkle

## Mac OS X

Install Qt:

  brew install qt

Install [Sparkle](http://sparkle.andymatuschak.org/) by moving _Sparkle.framework_ to /Library/Frameworks

# Build Instructions

  qmake
  make

The resulting binary can be found in the ``build`` subfolder.
